== Description ==

Secure your site!! No-one will be able edit/modify system files(theme+plugins) from Wordpress Dashboard,even admins (So, only from FTP can be edited)...   It is useful, when you share Admin access to others. IF PROBLEMS, JUST REMOVE PLUGIN.


>  (P.S.!!  This is a LIGHTWEIGHT (NON-SLOWING) Plugin.  See other <a href="http://bitly.com/MWPLUGINS" target="_blank" alt="http://codesphpjs.blogspot.com/2014/10/nsp-non-slowing-plugins-for-wordpress.html">MUST-HAVE PLUGINS</a> for everyone. )


== Installation ==

1. Install plugin from "ADD NEW" or Upload directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
